Política de Privacidade

Neon Idle Escape · Última atualização: 29/05/2026

Neon Idle Escape é um jogo gratuito desenvolvido por Samuel Silva. Esta política explica, de forma simples, como o aplicativo lida com dados, anúncios, compras dentro do aplicativo e recursos de personalização.

1. Dados salvos pelo jogo

O jogo pode salvar localmente informações de progresso, como moedas, diamantes, skins, upgrades, conquistas, configurações, recompensas e preferências de áudio. Esses dados são usados para manter o progresso do jogador dentro do aplicativo.

2. Anúncios

O jogo pode exibir anúncios por meio de redes de publicidade, como Google AdMob. Essas redes podem usar identificadores do dispositivo, dados técnicos e informações de uso para exibir, medir e melhorar anúncios, conforme as próprias políticas desses serviços.

Os anúncios premiados são opcionais. O jogador pode assistir a um anúncio para receber recompensas dentro do jogo, como moedas, diamantes, baús, troca de opções ou outros benefícios.

3. Compras dentro do aplicativo

O jogo pode oferecer compras dentro do aplicativo, como diamantes, moedas, pacotes, itens digitais ou benefícios opcionais. Essas compras são processadas por plataformas de pagamento compatíveis com a loja onde o aplicativo estiver disponível.

As recompensas de compras são entregues apenas após confirmação do pagamento pela plataforma responsável.

4. Imagem de perfil/avatar

Caso o jogador escolha uma imagem da galeria para usar como avatar, essa imagem será usada apenas para personalização dentro do aplicativo. O app não vende essa imagem nem a utiliza para fins externos.

5. Serviços de terceiros

O aplicativo pode usar serviços de terceiros, incluindo:

  • Google AdMob, para anúncios e anúncios premiados.
  • Serviços de faturamento da loja, para compras dentro do aplicativo.

Esses serviços podem possuir suas próprias políticas de privacidade e regras de tratamento de dados.

6. Compartilhamento e venda de dados

Não vendemos dados pessoais dos usuários. O aplicativo pode compartilhar dados técnicos limitados com serviços de anúncios ou faturamento apenas quando necessário para funcionamento, medição, segurança, prevenção de fraude ou processamento de compras.

7. Crianças e classificação

Neon Idle Escape é um jogo arcade casual. Caso crianças utilizem o aplicativo, recomenda-se supervisão de pais ou responsáveis, especialmente em relação a anúncios e compras dentro do aplicativo.
